62  This means that clock-switching is enabled.
63 
64  If you want to use the primary oscillator you must define:
65  #define POSC_NOMINAL xxxxULL
66  where xxxs is the frequency in hertz of the oscillator.
67  Also you must configure the divider in the Config Word2 to the
68  appropriate value so that the PLL block has as input frequency 4 MHz.
69  E.g. if your oscillator has 16MHz output you must use PLLDIV_4
